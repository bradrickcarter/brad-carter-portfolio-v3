Practical Typography Rules
✓  Body font-size: 15–16px minimum. Never below 15px for prose.
✓  Line-height: 1.2–1.45 for body text. Display/H1 = 1.15. Never exceed 1.45 for body.
✓  All-caps tracking: 0.08em on T.label. Never add letter-spacing to normal lowercase text.
✓  Text color: Never use pure #000 or #fff. Min #888 for small text on dark backgrounds.
✓  Heading hierarchy: Max 3 distinct sizes in any single view.
✓  Font consistency: T.mono for all preview panels and code. T.font for chrome only.
✓  Max line length: 65ch (T.maxW) for all prose blocks.
